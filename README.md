### 2. **PR Impact Analysis** (GitHub Integration)

**Automatically detect breaking changes in Pull Requests:**


**What it catches:**
- ‚ùå **Deleted functions** with existing references
- ‚ö†Ô∏è **Signature changes** that may break calls
- ‚úÖ **New functions** (safe additions)
- üîÑ **Renamed functions** needing updates

